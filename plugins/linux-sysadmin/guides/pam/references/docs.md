- `man pam` — overview
- `man pam.conf` — config file format
- `man pam.d` — per-service config directory
- `man limits.conf` — resource limits
- `man access.conf` — access control
- `man faillock` — account lockout management
- `man pwquality.conf` — password quality settings
- `man pam_unix` — standard Unix authentication module
