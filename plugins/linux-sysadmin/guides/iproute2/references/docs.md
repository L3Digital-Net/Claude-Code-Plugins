## Man Pages

- `man ip` — primary command reference
- `man ip-address` — address management
- `man ip-link` — link/interface management
- `man ip-route` — routing table management
- `man ip-neighbour` — ARP/neighbor cache
- `man ip-netns` — network namespace management
- `man ip-rule` — policy routing
- `man ip-tunnel` — tunnel management
