## Man Pages

- `man apparmor` — overview
- `man apparmor.d` — profile syntax
- `man aa-status` — show profile states
- `man aa-genprof` — generate profiles interactively
- `man aa-logprof` — update profiles from logs
- `man aa-enforce` / `man aa-complain` — mode switching
- `man apparmor_parser` — load/reload/validate profiles
